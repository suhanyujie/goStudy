## 原理
* 左子树和右子树的高度之差绝对值不超过1
* 而且规定，平衡二叉树的每个节点的平衡因子只能是-1 ，1 ，0；

### 普通二叉树
* 普通二叉树在极端情况下，有可能变成单链表。如果是单链表，对于二叉树来说，查找二叉树变成线性的，那就完全没有意义了
